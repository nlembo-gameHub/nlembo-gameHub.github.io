Lone Bowman

Project Type: University/Group Project

Role: Lead Programmer, Level Designer, Documentation

Software Used: GameMaker Studio 1.8

Languages Used: GameMaker Language

My first ever game created at Wake Technical Community College; Lone Bowman was a student project I and another student named Jonathan Lorenzo created using GameMaker Studio 1.8. The Project itself was my first step into game development and working in a team with others. I was the sole programmer and level designer while Jonathan Lorenzo created the art and both of us worked on quality assurance and game design. The game focuses on the player taking the role of an archer and fighting melee and ranged enemies in a forest, cave, and finally a gladiatorial arena. The player has access to different arrow power-ups and fights a balance of different enemies with the final one having special powers and skills.
